## Project Overview

Business Insights 360 is a comprehensive Power BI dashboard designed to provide a 360-degree analytical view of organizational performance across Finance, Sales, Marketing, Supply Chain, and Executive Management.

This project integrates cross-functional data into a centralized decision-support system, enabling leadership to monitor key performance indicators (KPIs), identify trends, detect inefficiencies, and drive strategic decision-making.
